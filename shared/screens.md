# 화면 설계서 - 대학 동문 SNS 플랫폼

> 디자인: 화이트/블랙 모던, 모바일 최적화
> 프레임워크: Next.js 14 App Router

---

## 라우팅 구조

```
/                     → 랜딩 (비로그인) / 피드 (로그인)
/login                → 로그인
/register             → 회원가입
/feed                 → 메인 피드
/posts/[id]           → 게시글 상세
/posts/write          → 게시글 작성
/members              → 회원 목록 (동문 검색)
/members/[id]         → 회원 프로필
/announcements        → 공지사항
/meetings             → 모임 목록
/meetings/[id]        → 모임 상세
/chat                 → 채팅방 목록
/chat/[roomId]        → 채팅방
/dm                   → DM 목록
/dm/[userId]          → DM 대화
/finance              → 회비/회계 대시보드
/notifications        → 알림
/settings             → 설정
/admin                → 관리 (회장/부회장)
/admin/approvals      → 가입 승인 관리
```

---

## 1. 랜딩 페이지 (/)

```
[로고: 동동 - 동문 네트워크]

대학 동문을 위한
소통 플랫폼

[카카오 로그인] [구글 로그인]
[이메일 로그인]
─────────────
아직 회원이 아니신가요? [가입 신청]
```

---

## 2. 로그인 (/login)

```
← 뒤로

로그인

[이메일 입력]
[비밀번호 입력]
[로그인 버튼]

── 또는 ──
[카카오 로그인] [구글 로그인]

[회원가입] [비밀번호 찾기]
```

---

## 3. 회원가입 (/register)

```
← 뒤로

가입 신청

Step 1: 기본 정보
[이메일] [비밀번호] [비밀번호 확인]
[이름] [전화번호]

Step 2: 동문 정보
[대학명] [학과] [입학년도] [졸업년도] [학번]

Step 3: 프로필 (선택)
[프로필 사진] [자기소개]
[회사] [직위] [거주지]

[가입 신청하기]
※ 회장 승인 후 이용 가능합니다
```

## 13. 회비/회계 (/finance)

```
← 뒤로

[요약 카드]

  총 잔액: 1,250,000원

[탭: 내 납부내역 | 회계장부]

── 내 납부내역 ──
[회비 카드]
  2024년 3월 월회비 | 30,000원
  납부일: 2024.03.05 | ✓납부
─────────────────
  2024년 2월 월회비 | 30,000원
  미납 | [납부하기]
─────────────────

── 회계장부 (총무 이상) ──
수입: 900,000원 | 지출: 450,000원
─────────────────
[내역 리스트]
  + 3월 회비 수입 | +900,000 | 03.10
  - 모임 장소 대관 | -200,000 | 03.08
─────────────────
```

## 하단 탭 네비게이션

```
[피드] [모임] [채팅] [회비] [더보기]
  📋      🤝     💬     💰     ≡
```

더보기 메뉴:
- 내 프로필
- 회원 검색
- 공지사항
- 알림
- 설정
- 관리자 (역할별 표시)

## 반응형 브레이크포인트

- Mobile: < 640px (기본)
- Tablet: 640px ~ 1024px (2열 그리드)
- Desktop: > 1024px (사이드바 + 메인 + 우측패널)
